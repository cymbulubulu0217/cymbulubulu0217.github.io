搭建环境

node.js

1、官网
最新版：node.js 中文网
旧版：当git上克隆项目后npm i报错时,原因时node版本太高了，需要我们降低版本
nodeJs 旧版本 寻找合适的项目版本后，点击下载，找到后缀名为 .msi的链接，就是当前node版本的安装包了
2、安装时，一直点击next就行
3、安装完毕后，cmd打开命令提示符，输入
node -v，
当提示
v16.15.0
时，node安装成功!
4、切换镜像源
全局切换淘宝镜像源：npm config set registry http://registry.npm.taobao.org
查看当前镜像：npm config get registry
5、下载yarn
yarn官网
下载：
npm install --global yarn,
通过如下命令测试yarn是否安装成功：
yarn --version
设置淘宝镜像源：yarn config set registr https://registry.npm.taobao.org

ps：镜像源更新了，请自行查看
用CNPM/淘宝源的开发者们请注意，淘宝NPM 镜像站喊你切换新域名啦。
新的Web 站点：https://npmmirror.com，Registry Endpoint：https://registry.npmmirror.com。
随着新的域名已经正式启用。
老 http://npm.taobao.org 和 http://registry.npm.taobao.org 域名将于 2022 年 05 月 31 日零时起停止服务。

2、git
1、安装
git官网
2、安装步骤：一路点next
3、检查git是否安装成功
git version,当出现git version 2.36.1.windows.1时，则说明安装成功
4、配置用户名邮箱


5、配置ssh

注意：ssh命令中-keygen前切勿加空格，否则报错

该命令会在目录/root/.ssh生成id_rsa私钥、id_rsa.pub公钥，接着用记事本打开id_rsa.pub公钥，把里面的内容添加到gitee和github的ssh配置里。
6、验证ssh key是否添加成功
输入ssh -T youemail,返回“Hi，successful….”说明配置成功。
